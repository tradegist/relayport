# Future Broker Relays

Candidate brokers to add as relay implementations alongside IBKR.

## Top Picks

### Alpaca
- Commission-free, REST + WebSocket, very developer-friendly
- Popular in algo-trading communities
- **Free paper-trading account** — no funding required, full API access with simulated money
- Best option for development without any commitment

### Tradier
- Built API-first, simple REST API, reasonable pricing
- Good for options too
- **Sandbox environment** available — sign up for API key, no funded account needed
- Market data in sandbox is delayed/simulated

### Tastytrade (formerly Tastyworks)
- Official API recently released, strong options-focused community
- Account creation required for API credentials (may require identity verification)
- Sandbox/cert environment available

### Saxo Bank
- OpenAPI, covers global markets (equities, FX, CFDs), big in Europe
- **Developer Portal** with simulation environment (no live account needed)
- Onboarding can be slower — they review applications
